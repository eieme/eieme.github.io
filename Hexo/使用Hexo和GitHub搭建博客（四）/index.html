在搭建 Hexo 博客过程中，无意间发现一个备份 Hexo 博客源文件的方法，于是就拿过来用了。-0-
由于 GitHub 的私有仓库是收费的，所以我们备份到 Coding 上。
这里用到的是 hexo-git-backup。
大家根据自己的实际情况参照 README.md 进行配置。

然后就是要实现自动备份功能，我想要在每次博客上传完成之后自动备份一次，可以监听 Hexo 的 deployAfter 事件。

修改 .gitignore 文件

如果没有请手动创建一个，在里面加入*.log 和 public/ 以及.deploy*/。因为每次执行hexo generate命令时，上述目录都会被重写更新。因此忽略这两个目录下的文件更新，加快push速度。
